## 📖 About

**FocusFlow** is a clean, minimal productivity web app that helps you manage your **tasks** and build lasting **habits** — all from a single, elegant interface. Powered by [Supabase](https://supabase.com/) for real-time authentication and data persistence.

---

## ✨ Features

### 📋 Task Manager
- Add tasks with **title**, **category**, **priority**, and **due date**
- Filter by category, priority, and status (Active / Done)
- Search tasks instantly
- Visual stats: Total, Completed, Pending, and Overdue counts with progress bars

### 🔁 Habit Tracker
- Create habits with custom **name**, **category**, **frequency**, and **emoji icon**
- Track habits by category: Wellness, Fitness, Mindset, Learning, Social
- Frequency options: Daily, Weekdays, or Weekends
- Visual stats: Total habits, Done today, Best streak, and Weekly completion rate

### 🔐 Authentication
- Email/password **Sign Up** and **Log In**
- Forgot Password support
- Secure session management via Supabase Auth

### 🌟 Other
- Daily motivational quote on the dashboard
- Toast notification feedback
- Fully responsive design
- Dark, modern UI with smooth interactions

## 🚀 Getting Started

### Prerequisites
- A [Supabase](https://supabase.com/) account and project
- A modern web browser

### Setup

1. **Clone the repository**
   ```bash
   git clone https://github.com/KavyaSreeBiji/focus_flow.git
   cd focus_flow
   ```

2. **Configure Supabase**

   Open `script.js` and replace the Supabase URL and anon key with your own project credentials:
   ```js
   const SUPABASE_URL = "https://your-project.supabase.co";
   const SUPABASE_ANON_KEY = "your-anon-key";
   ```

3. **Open in your browser**

   Simply open `index.html` in any modern browser — no build step required!
